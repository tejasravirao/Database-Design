Requirements to run this application

Softwares needed(for Windows):
1. MySQL Server
2. MAMP and MAMP PRO(4.0)
3. MySQL Workbench 


Procedure:
1. Create schema and populate tables as given for the assignment, in MySQL using the MySQL Workbench.
2. Start the MAMP Apache Server. Copy the .php project files into the htdocs folder in the MAMP root folder(C:\MAMP\htdocs).
3. Open Preferences in MAMP. Set Apache port to 8088.
4. On the browser type - localhost:8088/newpage1.php  .
